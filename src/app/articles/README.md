# Как добавить новую статью

1. Создайте новый файл в этой папке (`src/app/articles/data/`) с уникальным именем (например, `3.ts`).
2. Скопируйте и заполните следующий шаблон:

```ts
import type { ArticleData } from '../types';

const article: ArticleData = {
  slug: 'my-article-slug',
  title: 'Заголовок статьи',
  imageUrl: '/articles/my-article-image.jpg',
  content: 'Текст или HTML содержимое статьи.',
  datePublished: '2025-06-23',
  author: {
    name: 'Имя автора',
    type: 'роль или должность',
  },
  // SEO-описание статьи (опционально)
  seoDescription: 'Краткое описание для поисковых систем.',
  // SEO-ключевые слова через запятую (опционально)
  seoKeywords: 'ключевое слово 1, ключевое слово 2',
};

export default article;
```

3. Сохраните файл. Новая статья появится на сайте автоматически — функция [`getAllArticles`](../data/getAllArticles.ts) подхватит все файлы в этой папке, ручной импорт не требуется.

> **Рекомендация:** Изображения для статей размещайте в папке `/public/articles/` и указывайте относительный путь в поле `imageUrl`.

---

**Поля:**
- `slug` — уникальный идентификатор статьи (латиницей, используется в URL)
- `title` — заголовок статьи
- `imageUrl` — путь к изображению статьи
- `content` — содержимое статьи (строка, поддерживается HTML)
- `datePublished` — дата публикации в формате `YYYY-MM-DD`
- `author` — объект с информацией об авторе:
  - `name` — имя автора
  - `type` — роль или должность автора
- `seoDescription` — SEO-описание статьи (опционально, строка)
- `seoKeywords` — SEO-ключевые слова через запятую (опционально, строка)
